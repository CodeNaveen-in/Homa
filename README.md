# Homa
The Hospital Management System

A Project meant to experience and stimulate a hospital management system with full frontend, backend and database.

## Techstack 
- HTML
    - Jinja Template
- CSS
    - Bootstrap
- JS
    - Scroll reveal JS
- Python
    - Flask
- Sqlite

## Schema
- Has 3 Users 
    - Admin
    - Doctor
    - Patient

- has Tables : 
    - Users
    - Patient
    - Doctor
    - Department
    - Appointment
    - Treatment
    - and virtually craetes Medically History (Completed Appointment + Treatments)
